# 1. Успешная валидация email

## Тест-кейс: Валидный email с минимальной длиной локальной части

### Шаги воспроизведения
1. Ввести в поле ввода `k@academy.ru`.
2. Нажать кнопку «Проверить».

### Ожидаемый результат
Появился alert с текстом «Прекрасный email-адрес».

2. Нажать кнопку «Проверить».
3. Исправить значение на `test@mail.ru`.
4. Нажать кнопку «Проверить».

### Ожидаемый результат
